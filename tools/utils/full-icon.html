fullIcon()

This function is part of Iconify Utils package.

Function fullIcon() adds optional properties to IconifyIcon object.

Usage

Function has the following parameter:

  • icon, IconifyIcon. Icon data.

Function returns full icon data with type FullIconifyIcon.

Example

usage.ts
import { fullIcon } from '@iconify/utils/lib/icon';

const icon = fullIcon({
   body: '<path d="M7 6v12l10-6z" fill="currentColor"/>',
   width: 24,
   height: 24,
   hFlip: true,
});
console.log(icon);
Result:
{
   "left": 0,
   "top": 0,
   "width": 24,
   "height": 24,
   "rotate": 0,
   "vFlip": false,
   "hFlip": true,
   "body": "<path d=\"M7 6v12l10-6z\" fill=\"currentColor\"/>"
}
