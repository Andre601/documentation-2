defaults

This constant is part of Iconify Utils package.

Constant defaults is used for default FullIconifyIconCustomisations values.

Usage

import { defaults } from '@iconify/utils/lib/customisations';

const fullCustomisations = {
   ...defaults,
   height: 'auto',
};

Usually it is used with iconToSVG() function.
