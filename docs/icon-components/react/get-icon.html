Iconify for React function: getIcon

This tutorial is part of Iconify for React functions tutorial.

Function getIcon() retrieves icon data.

Usage

Function has the following parameter:

  • name, string. Icon name.

Function returns icon data in IconifyIcon format, null if icon is not available.

Examples

Code:
import { getIcon } from '@iconify/react';

const data = getIcon('bi:check2-circle');
