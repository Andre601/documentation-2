Icon components

Icon components are simple files that store icon data, one icon per file. Data is IconifyIcon object:

{
   "body": "<path d=\"M15 20a1 1 0 0 0-1-1h-1v-2h4a2 2 0 0 0 2-2V5a2 2 0 0 0-2-2H7a2 2 0 0 0-2 2v10a2 2 0 0 0 2 2h4v2h-1a1 1 0 0 0-1 1H2v2h7a1 1 0 0 0 1 1h4a1 1 0 0 0 1-1h7v-2h-7m-8-5V5h10v10H7z\" fill=\"currentColor\"/>",
   "width": 24,
   "height": 24
}

For more information about IconifyIcon type see IconifyIcon documentation.

Usage

Icon data is used in Iconify components:

  • Iconify for React.
  • Iconify for Vue.
  • Iconify for Svelte.

Icon data is passed as parameter to Iconify icon component, like this (example for React component):

// Import React
import React from 'react';

// Import Iconify component
import { Icon } from '@iconify/react';

// Import icon data
import homeIcon from '@iconify-icons/mdi-light/home';

// Sample functional component
export function MyComponent() {
   return (
       <div>
           <Icon icon={homeIcon} />
       </div>
   );
}

It can also be used in SVG framework using addIcon() function:

Iconify.addIcon('mdi:network-outline', {
   body:
       '<path d="M15 20a1 1 0 0 0-1-1h-1v-2h4a2 2 0 0 0 2-2V5a2 2 0 0 0-2-2H7a2 2 0 0 0-2 2v10a2 2 0 0 0 2 2h4v2h-1a1 1 0 0 0-1 1H2v2h7a1 1 0 0 0 1 1h4a1 1 0 0 0 1-1h7v-2h-7m-8-5V5h10v10H7z" fill="currentColor"/>',
   width: 24,
   height: 24,
});

Available icons

All icon sets available with Iconify API are available as standalone icon components. There are over 60,000 icons from more than 80 icon sets.

Packages

Putting so many files in one package would result in massive package, so icons are split into several NPM packages, two packages per icon set:

  • One package that uses modern ES6 export syntax: @iconify-icons/{prefix}.
  • One package that uses CommonJS module.exports syntax, which is used by Node.js: @iconify/icons-{prefix}.

where "{prefix}" is icon set prefix.

To import an icon, add icon name after package, for example, icon mdi-light:home can be imported from "@iconify-icons/mdi-light/home" (ES module) or "@iconify/icons-mdi-light/home" (CommonJS module)

It is recommended that you use ES6 module, recent versions of Node.js do support it. CommonJS are legacy packages.

TypeScript support

All icons in ES6 and CommonJS packages have .d.ts files, so they can be used with TypeScript.

Examples

Few examples of icon names:

  • mdi:login-variant from Material Design Icons can be imported from @iconify-icons/mdi/login-variant (ES6) or @iconify/icons-mdi/login-variant (CommonJS).
  • bi:chat-quote from Bootstrap Icons can be imported from @iconify-icons/bi/chat-quote (ES6) or @iconify/icons-bi/chat-quote (CommonJS).
  • tabler:info-circle from Tabler Icons can be imported from @iconify-icons/tabler/info-circle (ES6) or @iconify-icons/tabler/info-circle (CommonJS).
